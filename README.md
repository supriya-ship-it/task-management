# Quick Task Manager

A beautiful, fast task management web application with drag-and-drop functionality, undo support, and rich filtering options.

## Features
- ✅ 4 Categories: High Priority, TO DO, Sometime Later, Discuss
- 🖱️ Drag & Drop between categories
- 🔄 Undo functionality (Ctrl+Z)
- 🔍 Search by assignee name
- 👤 Hide/show assigned tasks
- ✏️ Edit tasks and assignees inline
- 💾 Local storage - your data persists
- 📱 Responsive design for all devices

## Usage
- **Add Task**: Type and press Enter
- **Edit Task**: Double-click task text
- **Edit Assignee**: Click on assignee name
- **Move Tasks**: Drag & drop or use dropdown
- **Search**: Type in search box to find tasks by assignee
- **Undo**: Press Ctrl+Z or click Undo button

## Deployment
This is a client-side application that runs entirely in your browser. Your data is stored locally using browser localStorage.

## Data Backup
Use the export/import functionality to backup your tasks or transfer between devices.
